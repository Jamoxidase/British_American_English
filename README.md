# British/American English Abliteration

Datasets and tools for abliterating American English patterns from LLMs to create British English models.

## Structure

- `british_american_data/` - Training datasets
  - `v1_large_dataset/` - 175 British + 176 American prompts
  - `v2_focused_dataset/` - 40 British identity-primed + 40 neutral prompts
- `heretic/` - Abliteration pipeline (forked from GitLitAF/heretic)
